ansible-role-ruby
=================

An Ansible Role that installs Ruby programming language (https://www.ruby-lang.org/en/).

Requirements
------------

This role only supports Ubuntu 12.x, 14.x, 16.x.

Role Variables
--------------

None.

Example Playbook
----------------

    - hosts: all
      roles:
         - { role: ansible-role-ruby }
